Case Study for Pokedex Project

Overview

To build a small web application with HTML, CSS, and JavaScript that loads data from an external API and enables the viewing of data points in detail.

Purpose and Context

Pokedex was a personal project I built as a part of my web development course at CareerFoundry. This is a simple Javascript Application that uses Fetch API to get details of Pokemon. We display the names of Pokemon and using eventListeners, we get the details of each Pokemon which opens up in a modal, by clicking on the Pokemon name.

Duration

It took me over 2 weeks to understand the concepts of JS, JQuery, DOM, API and to finish the project.

Approach

This is a simple Javascript Application that uses Fetch API to get details of Pokemon. We display the names of Pokemon.

Using eventListeners, we get the details of each Pokemon which opens up in a modal, by clicking on the Pokemon name. I have used Bootstrap for the Modal.
