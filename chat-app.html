Case Study for Chat App

Overview

To build a chat app for mobile devices using React Native. The app will provide users with a chat interface and options to share images and their location.

Purpose and Context

Chat App was a personal project I built as a part of my web development course at CareerFoundry. The app will be optimized for both Android and iOS devices. You will use Expo to develop the app and Google Firestore to store the chat messages.

Duration

It took me over 3 weeks to complete both the total project, as React Native is very new topic, It took me 15days just to know the concepts and terminology.

Features of the app:

  1. A page where users can enter their name and choose a background color for the chat screen before joining the chat.
  2. A page displaying the conversation, as well as an input field and submit button.
  3. The chat must provide users with two additional communication features: sending images and location data.
  4. Data gets stored online and offline.

Approach

It was developed with React Native, supported by Expo. I have installed Expo app in my phone for testing. I have used Gifted App to build my app components. There are mainly two screens for this app, which are Start screen and Chat screen. The start screen has options to enter your name and choose the background color. Then chat screen displays your name and there is a chat box, where you can send messages, and also can attach photos, use camera and even send location. Then I have used Firebase to store the messages. I have used Async storage to store the messages on client-side. Then I have used rendercustomActions component to get access to Camera, take Photo and to send Location.
