Case Study for MVS Portfolio

Overview

To build a portfolio web application with HTML, CSS, JavaScript, and React. The web application has different routes showcasing the photographers recent work, event page and his contact information.

Purpose and Context

MVS Portfolio is a project I built as a part of my freelancing job. Client requires a web page where he can showcase his work and his upcoming events. This is a simple React based single page application that uses React and React-Routing.

Duration

It took me over 1 week to get all the necessary requirements from the client and to finish the project.
