Case Study for Todo App

Overview

To build a web application using React-typescript. This web application is used to organise the daily tasks. The user can drag and drop the Tasks between active and completed containers.

Purpose and Context

Todo App is a project I built as a part of self learning React Typescript. The user can make a list of todo's. He can edit the todo's entered. He can strike them if they are completed and also can delete them. The main goal here is to drag the tasks and drop them in the completed container.

Duration

It took me over 2 weeks to learn the concepts of React-typescript and apply them for the project.

Approach

This is the todo page of the App. The user can add the tasks, and they gets added to the active container.

The user can edit the tasks he entered in the active container.

Once the task is completed, the user can drag the task and drop in completed container.
